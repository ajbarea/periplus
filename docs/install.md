# Install & Use

## Install on your iPhone (iOS 26)

1. Open the [planner](app/) in **Safari** (it's what creates a true standalone app on iOS).
2. Tap the **Share** button, then **Add to Home Screen**. Leave the "Open as Web App" toggle on (it's the default).
3. Tap **Add**. Periplus now has its own icon and opens full-screen.

!!! tip "Make offline durable"
    Open the app once while you have signal so the service worker can cache everything, and **install it to the Home Screen** (not just a bookmark). iOS keeps an installed web app's data instead of clearing it after a week of non-use.

## Install on Android
Chrome shows an **Install app** prompt, or use the ⋮ menu → **Add to Home screen**.

## Use it offline
Once cached, the entire itinerary (food, outdoor, the week-by-week plan, and your checklists) works with no connection. Directions links hand off to your maps app, which manages its own offline maps.
